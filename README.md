## 🔧 Featured Projects

### 🛡️ [Mem Watchdog](https://github.com/chf3198/crostini-mem-watchdog)
> VS Code extension + systemd daemon for ChromeOS Crostini — kills Chrome before the kernel OOM-kills VS Code

`earlyoom` hard-crashes on Crostini (`strtol()` overflow on the `SwapFree` sentinel value). This replaces it with a VS Code-aware watchdog daemon that tracks memory pressure via `/proc/meminfo` and PSI, manages `oom_score_adj` for all processes, and self-installs via the VS Code extension.
